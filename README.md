# Phonebook
Aplikacja wykorzystująca zewnętrzne API do fetchowania danych osobowych.
Po pobraniu rekordów jest możliwość zapisu rekordów lokalnie w pamięci telefonu, wraz z możliwością edycji zapisanych rekordów.
